Detailed Description

This is mostly a blend of step-12 and dflo code.

The problem to be solved is

\[ \frac{\partial \phi}{\partial t} \nabla \cdot (\phi \vec{v}) = 0 \]

\( \phi \) is the variable and \(\vec{v}\) is the advecting wind. The DG formulation of this problem is

\[ \int_{\Omega_h} l_j \left(\sum\phi_i l_i\right) \,d\Omega + \int_{\partial\Omega_h}l_j \left(\sum\phi^*_i l_i\right) \vec{v}\cdot\vec{n}\,dA - \int_{\Omega_h}\nabla l_j\cdot\vec{v} \left(\sum\phi_i l_i\right) \,d\Omega = 0 \]

  1. Uses numerical flux based approach, even for BCs (like in step-33)
  2. The global assembly is handled by MeshWorker

staticprivate

Integrates cell term.

Performs the integral

\[ \int_{\Omega_h} \left( l_i l_j + \nabla l_i \cdot \vec{v} l_j \right)\,dV \]

for a given cell. The above expression, multiplied by the vector of dof values gives the current cell term.

Note
This function is static because it doesn't depend on class instance variables
